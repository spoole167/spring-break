About this mini guide

This is one cut from Spring Break Cheat Sheets, the field guide to every breaking change between Spring Boot 3.5 and 4.0. Each card covers one change, verified by running real code on 3.5.16 and 4.0.7. One change, one card, one page: the error you'll see, the fix, and the traps.

How to read a card

  • Tier How it fails. Tier 1 won't build, Tier 2 won't start, Tier 3 behaves differently. Tier 3 is the dangerous one: nothing goes red, your data just changes shape.
  • Impact Effort per occurrence once found. S is find-and-replace, M needs an hour of care, L needs a plan.
  • OpenRewrite The badge means the Boot 4.0 recipe fixes this one for you. Run it before touching anything by hand.
  • What you'll see The real output, so you can match it against your own build.
  • The fix Before and after code, plus the steps when a rename isn't enough.
  • Watch out The traps that make this change look like a different bug.
  • Verify The command that proves you're done.
